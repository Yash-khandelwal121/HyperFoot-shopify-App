```
Time:    0s ──────── 10s ──────── 20s ──────── 30s
Position: ↗ ──────── ↙ ──────── ↗ ──────── ↙
Scale:    1.0 ────── 1.1 ────── 0.9 ────── 1.0
```

### Glow Pulse (Badges)
```
Time:     0s ──── 1s ──── 2s
Glow:     ◯ ──── ◯◯◯ ──── ◯
Intensity: 0.4 ── 0.6 ── 0.4
```

### Gradient Shift (Buttons)
```
Time:     0s ──── 1.5s ──── 3s ──── 4.5s
Position: ← ──── → ──── ← ──── →
```

```
Time:     0s ──── 0.4s ──── 0.8s
Opacity:  0% ──── 50% ──── 100%
Position: ↓ ──── ↓ ──── ↑
```
